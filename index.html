FAQ · 常問的幾件事

如果你想問……

「雞蛋糕」是誰？

是我女兒的小名。圓圓的、暖暖的、很可愛，所以叫雞蛋糕 🫶 這個網站上的大部分工具，都是在養育她的路上，一個一個為她做的。

Bookworm TW 是什麼？

一個 PM＋工程師媽媽 Nina，為女兒雞蛋糕做的繁體中文小工具集散地。做著做著發現，其他爸媽、其他孩子可能也會需要，所以就放在這裡分享。

這些服務要收費嗎？

目前全部免費使用、不用註冊。未來某些服務可能會有進階付費版本（例如更完整的資料、通知功能、更多城市支援），讓我能持續維護。基本功能會保持免費。

有建議、想回饋、或想跟雞蛋糕的媽媽聊聊？

歡迎寫信給我，或在各個服務的 GitHub repo 開 issue。一個人做的小攤，每一個回饋都非常珍貴。
